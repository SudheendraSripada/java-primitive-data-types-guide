Java Primitive Types Guide

Complete reference for Java's 8 primitive data types

1. Overview of Java Primitive Types

Java has 8 basic primitive types. They are called primitive because the values of these types are not objects in Java.

Type Size (bytes) Value Range Default Value
byte 1 -128 to 127 0
short 2 -32,768 to 32,767 0
int 4 -2,147,483,648 to 2,147,483,647 0
long 8 -9×10¹⁸ to 9×10¹⁸ 0L
float 4 -3.4×10³⁸ to 3.4×10³⁸ 0.0f
double 8 -1.7×10³⁰⁸ to 1.7×10³⁰⁸ 0.0d
boolean 1 true, false false
char 2 0 to 65,535 ''

2. Integer Types

int (Integer)

The most commonly used integer type. Used for whole numbers in everyday programming.

Range: -2,147,483,648 to 2,147,483,647
de>int n = "String".length();  // Returns length of string
int[] array = {1, 2, 3};
int size = array.length;     // Returns array length

long (Long Integer)

Used for very large integer values. Requires 'L' suffix or casting.

Range: -9×10¹⁸ to 9×10¹⁸
de>long bigNumber = 1000000000000L;  // Must use 'L' suffix
long result = 9223372036854775807L;

short (Short Integer)

Rarely used. Only when memory optimization is critical. 2 bytes.

Range: -32,768 to 32,767

byte (Byte Integer)

Smallest integer type. Most often used for storing large data blobs.

Range: -128 to 127
de>byte[] buffer = new byte[1024 * 1024];  // 1MB buffer for file I/O

3. Floating-Point Types

double (Double-Precision)

The standard floating-point type. Used in 99% of real number operations.

Size: 8 bytes | Precision: 15-17 significant digits
de>double pi = 3.14159265359;
double result = 1234567890.1234567;

float (Single-Precision)

Smaller than double but with less precision. Rarely used.

Size: 4 bytes | Precision: 6-7 significant digits
de>float value = (float) 123.456789;  // Results in 123.45679
float a = (float) 12345.9999;       // Results in 12346.0
Scientific Notation: Use E for exponents 
de>double a = 1.5E3;      // 1500.0
double b = 2.5E-2;     // 0.025

Infinity & NaN

de>System.out.println(100.0 / 0.0);     // Infinity
System.out.println(-100.0 / 0.0);    // -Infinity
System.out.println(0.0 / 0.0);       // NaN

4. char Type (Character)

Stores characters using Unicode encoding. Also a numeric type!

Size: 2 bytes | Range: 0 to 65,535
de>char a = 'A';              // Direct character
char b = 65;               // ASCII code for 'A'
char c = 0x41;             // Hexadecimal for 'A'
char d = 'A';         // Unicode for 'A'

Character Operations

de>char c = 'A';
c++;  // Now 'B'
System.out.println(c);     // Prints: B

Common Character Codes

Characters Codes
0-9 48-57
a-z 97-122
A-Z 65-90

5. boolean Type

The simplest primitive type. Only two possible values.

de>boolean isTrue = true;
boolean isFalse = false;
boolean result = (5 > 3);  // true

6. Type Conversion

Implicit Conversion (Widening)

Automatic conversion from smaller to larger type:

de>int num = 100;
long bigNum = num;         // Auto conversion
double decimal = bigNum;   // Auto conversion

Explicit Conversion (Casting)

Manual conversion from larger to smaller type:

de>double d = 123.45;
int num = (int) d;         // Results in 123 (loses decimals)
float f = (float) d;       // Results in 123.45
