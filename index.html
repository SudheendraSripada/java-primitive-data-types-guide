1. Encapsulation

Definition: Binding data + methods together while restricting direct access to data members.

Example

class Student {
    private String name;
    private int age;

    // Setter
    public void setAge(int a){
        if(a > 0) age = a;
    }

    // Getter
    public int getAge() {
        return age;
    }

    public void setName(String n){
        name = n;
    }

    public String getName(){
        return name;
    }
}

public class Main {
    public static void main(String[] args){
        Student s = new Student();
        s.setName("Arun");
        s.setAge(20);
        System.out.println(s.getName());
        System.out.println(s.getAge());
    }
}

Console Output (GFG Style)

Arun
20
Encapsulation protects data and avoids misuse of variables.

2. Inheritance

Inheritance allows a class to acquire properties of another class. Use extends.

Example

class Animal {
    void eat() { System.out.println("Eating..."); }
}

class Dog extends Animal {
    void bark() { System.out.println("Barking..."); }
}

public class Main {
    public static void main(String[] args){
        Dog d = new Dog();
        d.eat();
        d.bark();
    }
}

Output

Eating...
Barking...

3. Polymorphism

Same method name → different behaviors.

Method Overloading

class Maths {
    int add(int a, int b){
        return a + b;
    }
    double add(double a, double b){
        return a + b;
    }
}

Method Overriding (Runtime Polymorphism)

class Animal {
    void sound(){
        System.out.println("Some sound");
    }
}

class Cat extends Animal {
    void sound(){
        System.out.println("Meow");
    }
}

public class Main {
    public static void main(String[] args){
        Animal a = new Cat();
        a.sound();
    }
}

Output

Meow

4. Abstraction

Hiding complex implementation and showing only necessary details.

Abstract Class Example

abstract class Vehicle {
    abstract void start();
}

class Car extends Vehicle {
    void start(){
        System.out.println("Car starts with key");
    }
}

Interface Example

interface Shape {
    double area();
}

class Circle implements Shape {
    public double area(){
        return 3.14 * 5 * 5;
    }
}

Output

Car starts with key
78.5
